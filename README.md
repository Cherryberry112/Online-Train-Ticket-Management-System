###  Key Features

* **User Module**: Secure Registration, Login, Profile Management, and Password Recovery.
* **Booking System**: Real-time ticket search, seat selection, and digital payment integration (Card/Mobile Banking).
* **Live Support**: Integrated "Live Chat," Call Support, and direct email connectivity to Regional Managers.
* **Trip Management**: Live train tracking, route detail viewing, and travel history logs.
* **Feedback System**: Post-trip ratings, comments, and suggestions to improve service quality.
* **Admin Dashboard**: Advanced system reports, ticket management, and user oversight.

###  Technical Implementation

* **Platform**: Oracle APEX
* **Database**: Oracle SQL (DDL scripts included)
* **Modeling**:
* **ERD**: Complete Entity-Relationship Diagram (Standardized for 11g).
* **DFD**: Level 0, Level 1, and Level 2 Data Flow Diagrams for complex processes like "Manage Trips."
* **UML**: State Machine and Deployment diagrams to visualize system behavior and architecture.



###  Repository Structure

* **`Project final report.pdf`**: 100+ page comprehensive documentation covering SRS, analysis, and design.
* **`SQL_Tixfriend_BD.ddl`**: The complete SQL script used to generate the database schema (Tables, Constraints, Indexes).
* **`TixFriend_ER.dmd`**: Oracle SQL Developer Data Modeler file for the database architecture.
